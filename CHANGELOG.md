## [Unreleased]

### Added
- Initial release of TogoMQ SDK for PHP
- Client class for managing connections and operations
- Config class with builder pattern for configuration
- Message class for publishing and receiving messages
- SubscribeOptions class for subscription configuration
- Logger implementation with configurable log levels
- Comprehensive error handling with TogoMQException
- Publishing support via pubBatch() method
- Subscription support via sub() method using PHP generators
- Full unit test coverage
- GitHub Actions CI/CD pipeline
- PHP CS Fixer configuration for PSR-12 compliance
- PHPStan static analysis configuration
- Example scripts for publish and subscribe operations
- Comprehensive documentation (README, AGENTS.md, CONTRIBUTING.md)
